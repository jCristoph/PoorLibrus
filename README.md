UWAGA!
Aby serwer działał poprawnie po pobraniu projektu trzeba stworzyć bazę danych - w pliku serverTCP.cs w metodzie Start() postępować zgodnie z zamieszczoną tam instrukcją.
Następnie po stworzeniu bazy danych i uruchomieniu serwera można sprawdzić funkcje serwera logując się na przykładowe konta:
uczeń: LOGIN s10001 now123ak
nauczyciel: LOGIN t90001 Nauczyciel1
admin: LOGIN A99999 database123
